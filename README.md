# daily-posts
Write and read daily post

Use HTML, CSS, JS, NodeJS 
# Contents
POST **/upload** to upload new post

GET **/about**

GET **/contact** to contact us

GET **/posts/postTitle** to read specifical post

# How to install
**Clone from github:** 
`https://github.com/dinhtuan-zzz/daily-posts.git`

**Install package:** 
`npm install`

**Run project:** 
`npm start`
